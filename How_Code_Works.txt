========================================================
            HOW THE APPLICATION CODE WORKS              
========================================================

HIGH LEVEL WORKFLOW:
1. The user interacts with the React Frontend layout (buttons, forms).
2. The Frontend (using Axios or Fetch API) makes HTTP calls to the Spring Boot Backend.
3. The Spring Boot Backend receives the signal in its Controllers.
4. The Controller uses Repositories to interact with the MySQL Database.
5. The Database returns data to the Backend, which passes it back to the Frontend as JSON data.
6. Finally, the React Frontend re-renders the UI utilizing the new data.
